This repository contains the R code used for the simulation study in the paper:
"EM-Based Estimation for Regression Models of Temporally Ordered Count Data with Missing Responses"

## Contents
1. PoGM.R and PoGM (RB, MSE and Plots).R - Simulation code for Poisson Generalized Linear Model
3. PoGM.xlsx - Simulation results across different parameters and sample sizes of PoGM
2. NBGM.R and NBGM (MSE, RB and Plots)- Simulation code for Negative Binomial Generalized Linear Model
3. NBGM.xlsx- Simulation Results across different parameters and sample sizes for NBGM 

## Requirements
1. R version 4.3.1 or higher
2. Packages: ggplot2, dplyr, tidyr, AER
